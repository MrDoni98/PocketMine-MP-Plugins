#TimeLimitedBan
このプラグインは指定された時間だけBanすることができます。(もうありそうな気がしますがw)

##コマンド

| コマンド  | 説明  |
| ------------- | ------------- | ------------- |
| /tban <プレーヤー名> <時間(m)> [理由] | プレーヤーを指定された時間だけBanします  |
| /tban-ip <プレーヤー名\|IPアドレス> <時間(m)> [理由]  | プレーヤーを指定された時間だけIPBanします  |

※banの解除は通常通り/padron
